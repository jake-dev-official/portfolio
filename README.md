# Developer Portfolio

A responsive, high-performance professional portfolio built with React and Vite. This project features a custom-built UI, integrated contact systems, and optimized asset delivery via GitHub Pages.

🔗 **[Live Preview](https://jake-dev-official.github.io/portfolio/)**

---

## 🚀 Key Features

- **React-Powered UI** — Built with a component-based architecture for scalability and performance.
- **Formspree Integration** — Fully functional contact form with automated email delivery and spam protection.
- **Mobile-First Design** — Fluid layouts ensuring a seamless experience across mobile, tablet, and desktop devices.
- **Vite Optimized** — Extremely fast load times and optimized production builds.

---

## 🛠️ Tech Stack

| Category | Technology |
|----------|------------|
| **Framework** | React.js |
| **Build Tool** | Vite |
| **Styling** | CSS3 / Modern Flexbox & Grid |
| **Contact API** | Formspree |
| **Hosting** | GitHub Pages |

---

## 📂 Project Structure

- `/src` — Contains the React components, logic, and main application entry points.
- `/public` — Static assets (images, favicon) that are copied directly to the build.
- `/dist` — The production-ready build output hosted on the `gh-pages` branch.
